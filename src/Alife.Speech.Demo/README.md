# Alife Speech AI Demo

本模块演示了如何将语音识别（STT）与语音合成（TTS）接入大模型，实现纯语音的交互体验，并具备智能的外设感知能力。

## 核心功能

### 1. 硬件敏感监听 (Headphone Trigger)
- **隐私保护**：真央的语音识别功能**仅在插上耳机时**才会开启。
- **实时感知**：程序会实时监控 Windows 默认音频播放设备的变化。
- **状态同步**：识别开启/关闭时会通过 **Windows 原生通知** 提示用户，并在终端显示状态。

### 2. AI 语音助理
- **大模型驱动**：集成了 `OpenAIChatService`，真央具备逻辑理解与猫娘语回复能力。
- **端到端语音**：您说话 -> 真央识别 -> AI 回复 -> 真央语音播报。
- **精简回复**：AI 针对语音场景优化，回复保持简短（30字以内），更像真实的伴侣对话。

- `插上耳机`：Demo 通过 `SpeechService.StartRecognition()` 动态激活插件听力。
- `拔掉耳机`：自动调用 `StopRecognition()` 进入待机模式。

### 3. 插件测试底座
- **架构一致性**：本 Demo 不再手动拼凑组件，而是直接加载 `Alife.OfficialPlugins.SpeechService`。这使其成为测试语音功能在真央主程序中表现的最佳底座。

## 快速启动

1. **环境准备**：
   - 确保 `model` 目录下包含有效的 Vosk 模型。
   - 确保 Python 环境已安装 `edge-tts` (用于语音合成)。
2. **运行**：
   ```bash
   dotnet run
   ```
3. **交互**：插上耳机后开始说话，真央会通过系统通知和你打招呼喵！

---
*提示：如果无法弹出通知，请检查 Windows 设置中的“通知和操作”是否已开启。*
